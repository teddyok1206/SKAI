# Home Hero Copy Candidates

## Role

The home headline should not teach a prompt trick or advertise generic productivity. It should make SKAI's central claim visible:

> AI skill is the ability to define unclear reality, structure it into tasks, direct model work, verify the outputs, and leave the process as a reusable artifact.

The current line, `AI를 잘 쓰는 법을 문제로 연습한다.`, is clear but still sounds close to a general AI-use course. The replacement should move one step deeper toward orchestration.

## Selection Criteria

- Beginner-readable in the first three seconds.
- Clearly different from a prompt-tip site or chatbot wrapper.
- Points to problem definition, structure, direction, verification, or trace.
- Does not overuse myth, fire, genius, or model-performance language.
- Works with the existing lead: `문제 정의, task 분배, 자료 활용, 검증 흐름을 하나의 풀이로 남긴다.`

## Shortlist

### 1. 현실을 구조화하고, AI를 지휘하는 법을 훈련한다.

- Best default candidate.
- Captures real-world ambiguity and user agency.
- Strongly separates SKAI from prompt-template education.

### 2. 불명확한 문제를 AI와 풀 수 있는 구조로 바꾼다.

- Most beginner-friendly.
- Emphasizes transformation from messy reality to solvable structure.
- Slightly less explicit about orchestration and verification.

### 3. 프롬프트가 아니라, 문제 해결의 흐름을 설계한다.

- Strong philosophical contrast.
- Good if SKAI wants to reject prompt-trick framing very directly.
- Slightly more abstract for first-time users.

### 4. AI에게 답을 묻기 전에, 문제의 구조를 만든다.

- Sharpest anti-chatbot stance.
- Makes the cold-start discipline intuitive.
- Less complete because it focuses on the first phase more than the whole trace.

### 5. 의도, 자료, 검증을 엮어 AI 풀이를 만든다.

- Most native to SKAI's graph philosophy.
- Compact and artifact-oriented.
- Could feel slightly conceptual without the lead sentence.

### 6. AI 활용 능력은 답변이 아니라 흐름에서 드러난다.

- Strong for the Intelligence Mirror direction.
- Makes trace capture and process evaluation feel inevitable.
- Better for report/share surfaces than the first home headline.

## Recommendation

Use candidate 1 as the main home headline:

> 현실을 구조화하고, AI를 지휘하는 법을 훈련한다.

It is direct enough for first-time users, but still carries SKAI's non-negotiable philosophy: the user is not buying model magic or memorizing prompt tricks; the user is learning to structure intent, materials, tasks, and verification into a directed process.

Candidate 2 is the safest alternative if smoke-test users find candidate 1 too strong or unfamiliar.
